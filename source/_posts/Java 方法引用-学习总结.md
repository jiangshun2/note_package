# Java 方法引用学习总结

## 一、今日学习内容汇总

1. **方法引用的定义和语法**
   - 定义：复用已有方法作为函数式接口的实现
   - 语法格式：`::` 双冒号
   - 类型：静态方法引用、实例方法引用、构造方法引用、数组构造方法引用

2. **方法引用的使用条件**
   - 函数式接口：必须是只有一个抽象方法的接口
   - 方法签名匹配：参数类型、个数和返回类型一致
   - 上下文推断：编译器能从上下文推断类型

3. **方法引用的实际应用场景**
   - 集合操作：排序、遍历
   - Stream流：map、filter、forEach、collect
   - 函数式接口：作为Function、Consumer、Supplier等的实现

4. **方法引用与Lambda的关系**
   - 方法引用是Lambda的语法糖
   - 当Lambda体只是调用一个已有的方法时，使用方法引用更简洁

## 二、今日记忆口诀汇总

1. **方法引用基本概念**：方法引用双冒号，静态实例和构造；参数列表要匹配，函数接口是前提。

2. **方法引用使用条件**：函数接口是前提，方法签名要匹配；参数个数和类型，返回类型都得对；上下文要明确，编译器能推断。

3. **方法引用应用场景**：集合操作要排序，方法引用很方便；Stream流中常用到，map forEach最常见；函数接口作参数，方法引用更简洁。

4. **方法引用类型**：静态方法类名引，实例方法对象引；构造方法用new，数组构造类型引。
